Creador de HTML

HTML fue creado por el físico Tim Berners-Lee en 1989. El primer documento formal con la descripción de HTML, se publicó en 1991 con el nombre "HTML tags". Tim Berners-Lee lo define como:

HTML
"Lenguaje de marcado para la creación de documentos electrónicos que pueden ser leídos por humanos y máquinas"

Para más información, pinche aquí: Tim Berners-Lee
